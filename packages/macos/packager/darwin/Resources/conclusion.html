Registering your agent

Login to console.mondoo.com, select the appropriate space, then navigate to "Integrations", then click the + icon, and then select the "macOS" tile. Copy the Registration token and then register the agent by openning a terminal and running the following command:

  $ cnspec login --token {token}

Please login to the Mondoo Console

Uninstalling Mondoo

Run the following command to uninstall the Mondoo Client:

  $ sudo bash /Library/__PRODUCT__/uninstall.sh
