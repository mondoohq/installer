Mondoo Clients

Thank you for installing the Mondoo Platform Clients: cnquery and cnspec

We urge you to read the Mondoo Getting Started Guide for your next steps.

Registering your client

Login to console.mondoo.com, select the appropriate space, then navigate to "Integrations", then select the + icon, and then select the "macOS" tile. Copy the registration token and then register the clients by opening a terminal and running the following command:

  $ cnspec login --token {token}

Please login to the Mondoo Console

Uninstalling Mondoo

Run the following command to uninstall the Mondoo clients:

  $ sudo bash /Library/__PRODUCT__/uninstall.sh
